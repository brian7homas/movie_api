Returning Cast & Crew

  • The filmography object returns an array of objects that contain the title and url to each of a particular genre.
  • :genre parameter allows the user to choose what genre to return.
  • Returns a JSON object with the following values.
    •                             
                              "title":string,
                              "url":string,

The Filmography Object

*This is just an example object and will be updated soon.

This resource uses GET method to return a list of directors and actors

app.get('/content/movies/filmography/:title/:name', (req, res) => { let { title, name} = req.params; if(title === 'director'){ db.content.filmography.director.forEach(director=>{ if(director.name === name){ res.send(director) } }) }else if(title === 'actor'){ db.content.filmography.actor.forEach(actor=>{ if(actor.name === name){ res.send(actor) } }) } else{ res.send(db.content.filmography) } })
